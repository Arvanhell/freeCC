Kompendium Wiedzy: Element Iframe

Element iframe (inline frame) służy do osadzania innej zawartości HTML bezpośrednio na bieżącej stronie. Może to być wideo, mapa, inny element HTML, a nawet całe zewnętrzne strony internetowe.

Atrybut src określa adres URL strony, którą chcesz osadzić. Atrybuty width i height definiują wymiary ramki. Dobrą praktyką jest zawsze dodawanie atrybutu title, co jest kluczowe dla dostępności (accessibility).

Atrybut allow tworzy tzw. allowlist, która definiuje, co iframe może, a czego nie może robić (np. dostęp do schowka: clipboard-write). Atrybut allowfullscreen pozwala użytkownikowi na wyświetlanie zawartości w trybie pełnoekranowym.

Uwaga: Jeśli chcesz osadzić bezpośredni kod HTML zamiast zewnętrznego linku, musisz użyć atrybutu srcdoc zamiast src.
